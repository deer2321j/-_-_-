轻量模型依赖（编号1、编号3、编号7、v2）

v5一劳永逸模型依赖（编号2）
- 主要包：pandas、numpy、scikit-learn、catboost、joblib、tqdm、networkx、node2vec
- 安装示例：pip install pandas numpy scikit-learn catboost joblib tqdm networkx node2vec
- 说明：首次运行会训练文本管线与图嵌入（较慢）；后续加载缓存可在数分钟内生成结果。若本机 CPU 线程数较少，建议将脚本中的 Node2Vec 参数 workers 调低。

Kaggle环境模型依赖（编号4：dspos2、编号6：133）
- 推荐在 Kaggle 默认环境运行，仅需安装：torch_geometric
- 安装示例（Kaggle Notebook）：pip install torch_geometric
- 说明：Kaggle 提供免费 GPU，约半小时可完成复现；为保证与我们结果一致，强烈建议在 Kaggle 上运行。若在本地运行，需自行安装 PyTorch 与 torch_geometric，并按本机 CUDA/CPU 适配版本。

决赛 classic_tabular_autoML 依赖
- 该目录（classic_tabular_autoML - 超级无敌）提供 requirements.txt 与 README，按其说明安装（包含 AutoGluon Tabular 等）。
- 说明：该子项目计算量较大，建议在资源充足的环境，服务器运行。

融合脚本依赖（根目录）
